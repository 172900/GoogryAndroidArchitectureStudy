그 사이사이에 소소한 꿀팁, 노하우도 배울수 있어서 실무에서 잘 사용중인 기술, 정보들도 많았구요.
 덕분에 완전 초보였을때 아둥바둥 작업했었지만 지금은 나름에 아름다운 코딩을 하고 있습니다.
